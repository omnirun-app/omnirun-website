# Omnirun Website — Spec & Reference
# Last updated: April 9, 2026

## OVERVIEW

Marketing website for Omnirun desktop app.
Purely a brochure site — no user accounts, no dashboard, no auth, no blog, no FAQ.
Tech stack: Next.js 14 (App Router), Tailwind CSS, Vercel.
Repo: `D:\omnirun-website` (separate from main Omnirun app repo)

## HOMEPAGE

### Hero Animation Sequence

1. **Logo animation** (LogoAnimation.tsx — SVG embedded as React component, not iframe):
   - 200ms delay, then flash/blitz — CSS `drop-shadow` + `brightness` glow radiates from the logo
   - Glow uses 4 shadow layers: tight white core (15px), inner glow (50px), mid spread (120px), wide atmospheric (250px)
   - Brightness peaks at 13x — ring becomes pure white at peak
   - Ring fades in AFTER the flash peak (20%→50% of flash duration) — emerges from the light
   - Green dot pops in with bounce, orbits the ring
   - "omnirun" wordmark fades in as dot passes bottom
   - Final glow pulse — same curve as initial flash, logo radiates outward
   - Calls `onComplete` when finished

2. **Sequential tagline reveal** (after logo completes):
   - +200ms: **"Run everything."** appears (opacity only, no slide)
   - +700ms: **"Describe it."** appears (opacity only, no slide)
   - +1200ms: **"Done."** drops from above with bounce + subtle screen shake (max 1px)
   - "Done." is green (#2DB87A)

3. **Typewriter subtitle**: types at 25ms/char with blinking green cursor
   - "A desktop app that lets anyone build software, automate tasks, and control their computer — by simply describing what they want."

4. **"Join Waitlist" button** fades in after typewriter finishes — large, opens WaitlistModal

- Tagline: "Run everything. Describe it. Done."
- Alternative hero tagline: "Your computer finally listens."
- Currency: € (Euro)

- No code blocks in docs — plain English for non-technical users
- Testimonials framed as "Early Access Feedback" — honest, not suspicious
